# CanaryIslandTouristPRED

Predictive analysis and modeling of tourism data for the Canary Islands.

## Project Overview

This repository contains an analysis (and possibly predictive models) of tourism statistics in the Canary Islands (Spain). The main goal is to understand historical trends in tourist arrivals, stays, seasonality, nationality distributions, or similar metrics — and potentially forecast future tourism demand using statistical or machine learning techniques.

The core work seems implemented in an R Markdown document (`CanaryIslandTouristPRED.Rmd`):

- Data loading & cleaning
- Exploratory data analysis (EDA)
- Visualizations (time series, seasonal patterns, regional breakdowns)
- Statistical modeling / forecasting


## Technologies

- **Language**: R
- **Key packages** (most likely used): tidyverse (dplyr, ggplot2, tidyr), lubridate, forecast, prophet, knitr/rmarkdown, possibly others (caret/tidymodels if ML)
